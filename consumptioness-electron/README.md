# 🌐 Consumptioness — Electron

PC Power Consumption Calculator with TNB Malaysia tariff rates. Cross-platform desktop app built with Electron + React.

## Features
- One-click hardware scan (CPU, GPU, RAM, storage)
- Real-time power monitoring with live chart
- TNB Tariff A (Residential) progressive block pricing → RM
- Cost per hour / day / month / year
- Standby mode cost estimate
- Export to PDF & Excel
- Auto update via electron-updater + GitHub Releases

## Development
```bash
npm install
npm run electron:dev
```

## Build
```bash
npm run electron:build
```
